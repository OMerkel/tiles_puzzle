Rules

Slide the tiles to restore the original image. Click or tap a tile to move it towards the empty space.

Goal

Arrange all tiles in the correct order — left to right, top to bottom — leaving the empty space in the bottom-right corner.

Moving Tiles

Click any tile in the same row or column as the empty space to slide it. Arrow keys and WASD work too.

Line Shift

Hold Shift and press an arrow key (or W/A/S/D) to slide an entire row or column at once — counts as one move.

Keyboard Shortcuts
  • Arrow keys / WASD — move adjacent tile into the empty space
  • Shift + Arrow / WASD — shift an entire row or column
  • R — reshuffle the puzzle
  • ? — show shortcut reminder
Difficulty

Choose a grid size in Options: 3×3 (8-Puzzle), 4×4 (15-Puzzle), or 5×5 (24-Puzzle).

Options

Grid size and number overlay take effect immediately. Seed applies on next shuffle.
